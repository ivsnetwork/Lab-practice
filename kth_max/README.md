# 🔢 Problem: K-th Maximum Element (Without Sorting)

## 🧩 Problem Statement
Given an array of `n` integers and a number `k`, find the **k-th maximum element** in the array **without sorting the array**.

---

## 📥 Input
- An integer `n` — size of the array  
- An array `arr` of `n` integers  
- An integer `k`  

---

## 📤 Output
- The **k-th largest element** in the array  

---

## 🧪 Examples

### Example 1

Input:
n = 5
arr = 3 1 5 2 4
k = 2

Output:
4


---

### Example 2

Input:
n = 6
arr = 7 10 4 3 20 15
k = 3

Output:
10


---

## ⚙️ Constraints
- `1 ≤ n ≤ 10^5`
- `1 ≤ k ≤ n`
- `-10^9 ≤ arr[i] ≤ 10^9`

---

## 📌 Notes
- Do NOT sort the array  
- You must determine the k-th maximum using an efficient approach  
- Duplicate values are allowed  

---

## 🚀 Task
Implement a function to return the **k-th largest element** without using sorting.
